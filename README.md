# Mini Message Board

A simple message board built with Node.js, Express, and EJS.

## Features

- View all messages on the main page
- Add new messages via a form
- View individual message details

## Stack

- Node.js
- Express
- EJS

## Getting Started

Clone the repo and install dependencies:

\`\`\`bash
git clone https://github.com/ICELUV0x/node-message-board
cd node-message-board
npm install
\`\`\`

Run the server:

\`\`\`bash
node app.js
\`\`\`

Open `http://localhost:4044` in your browser.

## Notes

Messages are stored in memory — they reset every time the server restarts. No database yet.
